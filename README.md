# BotZakazov
# 0.перед запуском бота установите библиотеки коммандами: 
# ```pip install openpyxl==3.1.2```
# ```pip install python-telegram-bot==20.7``
# 1.Создатель бота: https://t.me/AyuGramXacker
# 2.Для запуска бота скачайте файл Main.py и поместите в директорию удобную для Вас. Перейдите в директорию с этим файлом в командной строке Windows/Linux (cd 'путь к директории в которой лежит файл') и введите python Main.py, он создаст остальные файлы для работы бота.
# 3.Настройки бота: если вы хотите поставить бота на свой токен измените переменную token (строка 586) на свою, как получить свой токен указано в пункте 4
# 4.Для получения токена бота, нужно иметь аккаунт в телеграмм и перейти по ссылке: https://t.me/BotFather в бота BotFather, вести /start, затем /newbot и следовать инструкции бота
# 5.При запуске файла Main.py создаются все нужные файлы. Если необходимо изменить список товаров, нужно в коде бота (Main.py) в функции _ensure_products_file(self) (строка 35) изменить список (конструкция указана в комментарии)
# 6.Ссылка на бота без измены токена: https://t.me/pzkhyper_bot
