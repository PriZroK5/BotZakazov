# BotZakazov
1.Создатель бота: https://t.me/AyuGramXacker
2.Для запуска бота скачайте файл Main.py перейдите в директорию с этим файлом в командной строке Windows/Linux (cd 'путь к директории') и введите python Main.py(желательно скачивать файл в созданную вами папку), он создаст остальные файлы для работоспособности.
3.Настройки бота: если вы хотите поставить бота на свой код измените переменную token (строка 586) на свою, как получить свой токен указано в пункте 4
4.Для получения токена бота, нужно иметь аккаунт в телеграмм и перейти по ссылке: https://t.me/BotFather в бота BotFather, вести /start, затем /newbot и следовать инструкции бота
5.При запуске бота бот создаст все нужные файлы (изменить список товаров можно в коде бота в функции _ensure_products_file(self) (строка 35))

Приятного использования!
